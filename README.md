# Diabetes Prediction using Machine Learning

## Course
CSCI3234 –ML

## Author
Md Sha Niamatullah

## Project Overview
This project implements a complete machine learning workflow:

- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Model Training
- Model Evaluation

Dataset: CDC Diabetes Health Indicators (UCI Repository)

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- Support Vector Machine
- XGBoost

## Evaluation Metrics
- Precision
- Recall
- F1 Score
- ROC-AUC
- PR-AUC
- Confusion Matrix
- ROC Curve
- Precision-Recall Curve

## Tools
Python, Scikit-learn, XGBoost, Google Colab

## Result
Logistic Regression achieved the best ROC-AUC performance among evaluated models.
